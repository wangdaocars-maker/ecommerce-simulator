# 功能测试指南

## 头部组件测试

### 测试步骤

1. **启动开发服务器**
   ```bash
   npm run dev
   ```
   访问: http://localhost:3002

2. **登录系统**
   - 使用测试账号登录:
     - 学生账号: `student1` / `123456`
     - 教师账号: `teacher` / `teacher123`
     - 管理员账号: `admin` / `admin123`

3. **测试头部功能**

   **左侧部分:**
   - ✅ 跨境卖家中心 Logo 显示正常
   - ✅ 店铺信息显示用户名称
   - ✅ 点击店铺名称，下拉显示当前用户信息和角色

   **右侧部分:**
   - ✅ 购物车图标显示（蓝色）
   - ✅ 通知铃铛图标显示
   - ✅ 用户头像显示（红色背景，显示姓名首字母）

   **交互测试:**
   - ✅ 鼠标悬停图标时颜色变深
   - ✅ 点击用户头像，下拉菜单显示：
     - 个人信息（暂无跳转）
     - 账号设置（暂无跳转）
     - 退出登录（点击后退出到登录页）
   - ✅ 点击"退出登录"，成功退出并跳转到登录页

### 预期效果

- 头部始终固定在页面顶部
- 背景纯白色，底部带 1px 灰色边框和轻微阴影
- 所有图标和文字大小、颜色、间距符合速卖通风格
- 退出登录后重定向到 `/login` 页面

### 已知问题

- 购物车功能未实现（仅显示图标）
- 通知功能未实现（仅显示图标）
- 个人信息、账号设置页面未创建

## 侧边栏组件测试

### 测试步骤

1. **登录后查看侧边栏**
   - 左侧显示 220px 宽的侧边栏
   - 背景色为浅灰白色
   - 包含12个主菜单项

2. **测试商品菜单展开/收起**
   - ✅ 默认状态：商品菜单已展开，显示7个子菜单
   - ✅ 点击商品主菜单，可以收起子菜单
   - ✅ 再次点击，可以重新展开
   - ✅ 展开状态：主菜单背景变为浅蓝色，箭头向上
   - ✅ 收起状态：主菜单背景变回白色，箭头向下

3. **测试子菜单跳转**
   - ✅ 点击"商品管理"，跳转到 `/products` 页面
   - ✅ 点击"商品发布"，跳转到 `/products/create` 页面
   - ✅ 当前页面在侧边栏中高亮显示（蓝色文字 + 浅蓝背景）
   - ⚠️ 其他子菜单暂无跳转（等待后续开发）

4. **测试红点徽章**
   - ✅ "商品"菜单右上角显示红点
   - ✅ "推广"菜单右上角显示红点
   - ✅ "资金"菜单右上角显示红点
   - ✅ "AI素材工具"子菜单右侧显示红点

5. **测试其他菜单项**
   - ✅ 首页菜单可点击，跳转到 `/` 页面
   - ⚠️ 其他菜单项（交易、物流、店铺等）显示为禁用状态，不可点击
   - ✅ 禁用菜单项有透明度效果，鼠标变为 `not-allowed`

### 预期效果

- 侧边栏固定在左侧，与头部完美对齐
- 商品菜单展开/收起动画流畅
- 当前页面在侧边栏中正确高亮
- 所有样式与速卖通原版一致

### 已知限制

- 除"商品发布"和"商品管理"外，其他子菜单暂无页面
- 除"首页"和"商品"外，其他主菜单为禁用状态（培训模拟器仅需商品管理功能）
